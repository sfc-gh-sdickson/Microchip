# Agent Creation SQL Script (ST)

Creates the Microchip Intelligence Agent for the ST environment with all tools (Cortex Analyst, Cortex Search, ML procedures).

## Quick Start

```sql
@HOL/ST/sql/agent/08_create_intelligence_agent.sql
```

## Verify Objects First
- Semantic views exist in `MICROCHIP_INTELLIGENCE_ST.ANALYTICS_ST`
- Search services exist in `MICROCHIP_INTELLIGENCE_ST.RAW_ST`
- Procedures exist in `MICROCHIP_INTELLIGENCE_ST.ANALYTICS_ST`
